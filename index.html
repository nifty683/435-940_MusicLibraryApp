Introduction

This web application is dedicated to assisting in created a moving plan. The basic capabilites of this web application should be sufficient for most users. This application was developed specifically for the Carbondale and greater Southern Illinois area.

Instructions

You can use the navigational tabs above to look around our site. To learn about the site and the developers behind it, go to the ABOUT tab. To read the instructions on creating a moving plan, go to the INSTRUCTIONS tab. To begin planning, go to the PLANNER tab. To contact us for any reason, go to the CONTACT tab.
